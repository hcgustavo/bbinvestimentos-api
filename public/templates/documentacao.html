Documentação da BB Investimentos API

Informações

  • Esta API é apenas para consumo. Assim, somente o método HTTP GET está disponível.
  • Não é necessário realizar nenhuma autenticação. Todos os recursos estão abertamente disponíveis.
  • A BB Investimentos API foi desenvolvida de forma independente. O Banco do Brasil não tem nenhum envolvimento com o projeto.

Política de uso

A BB Investimentos API é livre e todos podem utilizá-la. Pedimos apenas que todos os desenvolvedores sigam as seguintes regras:
  • Armazene localmente os recursos que você utilizar.
  • Todos os recursos possuem um timestamp. Dessa forma, use-o para saber quando houve modificações em um determinado recurso.
  • Os recursos são atualizados uma vez por dia. Portanto, não há necessidade de sobrecarregar o servidor com chamadas repetitivas.

Recursos

Todos os fundos

GET api/fundos/

Obtém uma lista com todos os fundos

Informações retornadas:

Nome Descrição Tipo de dado
total Quantidade total de fundos de investimento Inteiro
resultados Lista contendo nome, id e código dos fundos Lista: nome (String), id (Inteiro) e código (String)

Fundo específico

GET api/fundos/{id ou código}

Obtém um único fundo. Pode-se usar o id numérico ou código alfa-numérico do fundo.

Informações retornadas:

Nome Descrição Tipo de dado
id ID do fundo Inteiro
codigo Código alfa-numérico do fundo String
nome Nome do fundo String
prazoCarteira Prazo da carteira (curto, médio ou longo prazo) String
categoria Categoria do fundo (renda fixa ou renda variável) String
rentabilidade Rentabilidades (%) do dia, mês, mês passado, ano; e ainda últimos 12, 24 e 36 meses Decimal
plMedio12Meses Patrimônio líquido do fundo (em milhões R$) nos últimos 12 meses Decimal
taxaAdm Taxa de administração do fundo (%) Decimal
historicoCotacao Histórico de todas as cotações do fundo até o momento (datas e valores) Lista: data (String) e valor (Decimal)
info Contém várias informações do fundo. Consulte a tabela abaixo para ver todos os dados contidos em info (Ver tabela abaixo)
createdAt Timestamp que marca quando o fundo foi criado no banco de dados Timestamp
updatedAt Timestamp que marca quando o fundo foi modificado no banco de dados Timestamp

Dados contidos em info:

Nome Descrição Tipo de dado
descricao Descrição textual do fundo String
aplicacaoInicial Valor da aplicação inicial Decimal
aplicacoesSubsequentes Valor das aplicações subsequentes Decimal
resgate Valor para resgate Decimal
saldoMinimo Saldo mínimo Decimal
cotaAplicacao Cota de que dia será considerada na aplicação String
cotaResgate Cota de qual dia será considerada no resgate String
creditoContaCorrente Quando se dará o crédito em conta corrente após resgate String
horarioLimiteMovimentacao Horário limite para movimentação do fundo String
aplicacaoResgateProgramado Indica se fundo permite aplicação e resgate programados Boolean
opcaoResgateAutomatico Indica se fundo permite resgate automático Boolean
grauRisco Grau de risco do fundo (baixo, médio ou alto) String
dataInicioFuncionamento Data de início do funcionamento do fundo String
taxaSaida Taxa de saída Decimal
taxaPerformance Taxa de performance Decimal
categoriaANBIMA Categoria do fundo de acordo com a AMBIMA String
