Sobre a BB Investimentos API

O que é uma RESTful API?

Uma RESTful API define uma coleção de funções que podem ser utilizadas para realizar requisições e receber respostas via protocolo HTTP. Dessa forma, desenvolvedores podem interagir com aplicações de terceiros para obter informações que serão utilizadas em suas próprias aplicações.

No caso da BB Investimentos API, os desenvolvedores podem interagir com um banco de dados que contém dezenas de informações sobre mais de 40 fundos de investimento do Banco do Brasil.

Quais informações posso obter e em qual formato?

São cerca de 30 dados que podem ser obtidos de cada fundo. Alguns deles são:

  • Nome do fundo
  • Categoria
  • Rentabilidade (diária, mensal e anual)
  • Histórico de cotações
  • Taxa de administração
  • Grau de risco
Para a lista completa consulte a documentação da BB Investimentos API.

Ao realizar uma requisição com sucesso, o desenvolvedor receberá uma resposta com os dados de um fundo em formato JSON. Dessa forma, ele poderá facilmente acessar essas informações e incorporá-las em suas aplicações, independente da linguagem que estiver usando.

De onde vem todos os dados?

Todos os dados estão publicamente disponíveis no próprio site do Banco do Brasil. A nossa API simplesmente reune todos eles de forma estruturada em um banco de dados para que possam ser facilmente utilizados por todos os desenvolvedores interessados.

Quem é o responsável pelo desenvolvimento?

A BB Investimentos API foi criada por Gustavo Henrique Caixeta como um projeto pessoal. O Banco do Brasil não está envolvido de nenhuma forma com esse projeto.

Quais tecnologias estão envolvidas no desenvolvimento?

  • Implementação da API: Node.js, Express e Mongoose
  • Banco de dados: MongoDB
  • Web Front-end: AngularJS
  • Hospedagem da aplicação: Heroku
